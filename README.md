## Uploading your own files here
Create a pull request!
- Fork this repo and then come back to it.
- Locate and enter the directory you would like the files in (e.g. music for audio files)
- Click 'add file' in the top right corner and select 'upload files'
- Drop or select your file(s) that you want to upload.
- Commit the changes and wait for me to accept the change.
